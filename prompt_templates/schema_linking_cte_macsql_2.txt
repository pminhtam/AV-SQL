A common table expression, or CTE, is a temporary named result set created from a SELECT statement that can be used in a subsequent SELECT statement. Each SQL CTE is like a named query, whose result is stored in a virtual table (a CTE) to be referenced later in the main query.
CTE have format like:
```cte
WITH cte_name AS (
    SELECT column1, column2
    FROM table_name
    WHERE condition
),
cte_name2 AS (
    SELECT column1, column2
    FROM table_name2
    WHERE condition2
)
```
